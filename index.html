jquery-toaster by laurie71

A jQuery plugin that shows stacked transient notification messages.

  • Progressive enhancement allows toasts to be read when JavaScript is disabled
  • Support for *info*, *notice*, *warning* and *error* notices
  • Straight-forward API: `$(selector).toast()` to get started
  • Easily tailored through options and events
  • Fully customizable

Dependencies

jQuery Toaster requires jQuery 1.5 or later. Older versions may work, but have not been tested.

Install

Copy jquery.toaster.js, jquery.toaster.css> and the images> directory into your project, then link the JavaScript and CSS files into your page.
